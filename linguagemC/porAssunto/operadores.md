## Operadores Aritméticos

| Operador | Finalidade  | Exemplo | Resultado |
|   :-:    |     :-:     |   :-:   |    :-:    |
|   ``` + ``` |  Adição                     | ``` 5 + 2 ```  | ``` 7 ```  |
|   ``` - ``` |  Subtração                  | ``` 5 – 2	```  | ``` 3 ```  |
|  ``` * ```	|  Multiplicação	            | ``` 5 * 2	```  | ``` 10 ``` |
|  ``` / ```	|  Divisão (Quociente)	      | ```5 / 2 ```	 | ``` 2 ```|
| ``` % ```	  |  Divisão Euclidiana (Resto)	| ``` 30 % 7 ``` | ``` 2 ``` |
