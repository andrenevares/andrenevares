# Tipos de Variáveis em C

|  Tipo     |   o que é      |
|  :-:     |  :-:       |
|char|   |
|unsigned char|  |
|int|  |
|unsigned int| |
|short int| |
|long| |
|unsigned long| |
|float| |
|double| |
|long double|   |


Exemplos:

- ``` float n; ``` declara a variável ```n``` como uma variável do tipo float;
- ``` float n1, n2; ``` declaramos duas variáveis na mesma linha, a variável ```n1``` e a variável ```n2``` como tipo ```float```;
