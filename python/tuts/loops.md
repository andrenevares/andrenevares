## For Loops com Lists

```
times = ["Fluminense", "Flamengo", "Vasco", "Botafogo"]

for time in times:
  print(time)

```

## For Loops com range
- Não imprime o último número

```
>>> for x in range(5, 10):
...     print(x)
...
5
6
7
8
9
```

## While Loops 

```
idade = 0

while idade <= 5:
  print idade
```
