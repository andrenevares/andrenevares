# Crispy forms

Vai permitir que o utilizemos poucos códigos de modo a estilizar o nosso formulário alinhado com o bootstrap.

## instalar crispy forms

```python
pip intall django-crispy-forms
```

## colocar o crispy forms no settings

``` python
INSTALLED_APPS = [
  (...)
  'crispy_forms',
  (...)
]
```
